# eslint-config-gemcook

## Usage

### Install:

#### `npm`

```sh
npm install --save-dev \
  eslint-config-gemcook \
  eslint-config-fbjs \
  eslint-config-prettier \
  eslint-plugin-babel \
  eslint-plugin-import \
  eslint-plugin-flowtype \
  eslint-plugin-jsx-a11y \
  eslint-plugin-react \
  eslint-plugin-relay \
  eslint-plugin-prettier \
  eslint \
  babel-eslint \
  prettier-eslint
```


#### `yarn`

```sh
yarn add --dev \
  eslint-config-gemcook \
  eslint-config-fbjs \
  eslint-config-prettier \
  eslint-plugin-babel \
  eslint-plugin-import \
  eslint-plugin-flowtype \
  eslint-plugin-jsx-a11y \
  eslint-plugin-react \
  eslint-plugin-relay \
  eslint-plugin-prettier \
  eslint \
  babel-eslint \
  prettier-eslint
```

### Configure

Then create a file named .eslintrc with following contents in the root folder of your project:

```js
{
  "extends": "gemcook"
}
```
